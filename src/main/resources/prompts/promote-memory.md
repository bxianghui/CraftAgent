根据本次完整对话和已有长期记忆，输出最终的记忆列表。

规则：
1. 从对话中提取值得跨 session 保存的信息，生成新条目
2. 与已有记忆相同主题的条目合并为一条，保留最新最完整的内容
3. 已被新信息覆盖或不再有价值的旧条目直接丢弃（不在输出中出现即为删除）
4. 没有任何值得保存的信息时返回 []

type 可选：
- user: 用户背景信息、技术栈、偏好
- feedback: 用户对 AI 行为的反馈和偏好
- project: 项目状态、架构决策、重要信息
- reference: 外部资源、文档地址
- task: 进行中的任务状态

只输出 JSON 数组，不要任何解释：
[{"name":"xxx","description":"one-line description","type":"xxx","content":"详细内容"}]
